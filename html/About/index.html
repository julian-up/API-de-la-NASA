Sobre este proyecto

NASA APOD Explorer es una aplicación web interactiva que consume la API de la NASA para mostrar la imagen astronómica del día.

Este proyecto permite explorar imágenes astronómicas históricas, conocer su explicación científica y guardar tus favoritas en el navegador.

Tecnologías utilizadas

  • HTML5
  • CSS3
  • BOOTSTRAP
  • JavaScript
  • LocalStorage
  • NASA APOD API
